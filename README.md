# efficient_graph_algorithms
Implementations of efficient graph algorithms

## Installation
```bash
git clone git@github.com:Topographers/efficient_graph_algorithms.git
cd efficient_graph_algorithms
pip3 install -r requirements.txt
pip3 install -e . --user
```
